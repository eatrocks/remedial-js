Grunt setup
  -package.json describes my application including dependencies
  -so you need to run npm install to pull in the dependencies
  -npm install creates a node_modules folder to store the dependencies

Gruntfile.js configures grunt, a nodeJS task runner
  -I used this to automatically run the script I'm working on when saved
  -uses the grunt-execute plugin to run the changed script with node
  -uses the grunt-contrib-watch plugin to watch for changes
